# Dev-seed fixtures

Small, committed assets used by the local development seed (`npm run dev:seed`).
Unlike `samples/videos/` and `artifacts/` (both git-ignored), these are checked
in so the seed is reproducible on any clone with no extra downloads.

- **`demo-sprint.mp4`** — H.264 (High, yuv420p, `+faststart`), 1044×596, ~6.2 s.
  A browser-universal transcode of the local `samples/videos/test.mp4` (HEVC),
  produced by `npm run sample:transcode`. H.264 so it plays reliably in Chrome.
- **`demo-sprint.pose.json`** — the aligned MediaPipe `PoseSequence` for that
  clip (same dimensions), generated by the real analysis pipeline
  (`npm run mediapipe:video` / `worker:analysis`). Powers the interactive
  overlay, calibration, PB prediction, sprint phases, and sprint intelligence.

Regenerate only if the source clip changes: re-run `npm run sample:transcode`
(video) and re-run the MediaPipe pipeline over `test.mp4` (pose), then copy the
resulting `.pose.json` here as `demo-sprint.pose.json`.
